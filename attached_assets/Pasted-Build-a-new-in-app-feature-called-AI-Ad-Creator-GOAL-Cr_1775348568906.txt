Build a new in-app feature called **"AI Ad Creator"**.

GOAL:
Create a clean, premium, and simple feature that allows small businesses to generate high-converting flyers, posters, and social media ads using AI.

This is NOT a design tool like Canva.
This should feel like a guided “get customers fast” system.

---

## CORE USER FLOW:

1. User selects ad goal
2. User fills out a short form
3. App generates ad copy using AI
4. App generates 4 ad design variations using AI images
5. User refines one design with simple buttons
6. User downloads or exports the final design

Keep everything extremely simple, clean, and premium.

---

## BACKEND LOGIC:

STAGE 1 — GENERATE AD COPY

POST /api/ad-copy

Input:
{
businessName,
service,
location,
offer,
cta,
tone,
goal
}

Use AI to generate:

Return JSON:
{
"headline": "",
"subheadline": "",
"offer_line": "",
"cta": "",
"supporting_bullets": ["", "", ""],
"design_direction": "",
"image_direction": ""
}

---

STAGE 2 — BUILD IMAGE PROMPTS

Convert the copy into a structured image prompt.

Base template:

"Create a premium advertising flyer/poster for a local business.

Business type: {{industry}}
Service: {{service}}
Business name: {{businessName}}

Headline: {{headline}}
Subheadline: {{subheadline}}
Offer: {{offerLine}}
CTA: {{cta}}

Include supporting text:
{{bullet1}}, {{bullet2}}, {{bullet3}}

Design requirements:

* clean premium layout
* modern ad design
* strong hierarchy
* minimal clutter
* high readability
* professional marketing look
* no mockups
* no distorted text
* no messy layouts
* flat finished graphic

Format: {{format}}"

---

STAGE 3 — GENERATE 4 VARIATIONS

Create 4 styles:

1. Clean Professional
   "clean, minimal, trustworthy, premium service ad"

2. Bold Offer
   "high contrast, strong CTA, bold headline, still professional"

3. Luxury Premium
   "elegant, refined, high-end brand look"

4. Social Ready
   "modern, eye-catching, mobile optimized"

Call image generation 4 times.

---

STAGE 4 — REFINE

POST /api/ad-refine

Input:
{
generationId,
action
}

Actions:

* more-premium
* more-bold
* simplify
* improve-headline
* add-urgency
* change-colors
* resize

Modify prompt and regenerate image.
